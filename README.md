# perito-digital-lab

Proyecto para construir una marca digital discreta orientada a investigación digital, análisis forense, OSINT, trazabilidad tecnológica y documentación técnica/pericial.

## Estructura

- `app/`: web principal
- `content/blog/`: artículos y borradores SEO
- `content/videos/`: guiones, storyboard y subtítulos
- `docs/branding/`: posicionamiento, naming y propuesta de valor
- `docs/legal/`: notas legales y límites del proyecto
- `docs/research/`: ideas, temas y líneas editoriales
- `assets/`: recursos visuales
- `scripts/`: automatizaciones futuras

## Objetivo inicial
Validar demanda, construir autoridad y preparar una fábrica de contenido anonimizado a partir de conocimiento técnico reutilizable.
