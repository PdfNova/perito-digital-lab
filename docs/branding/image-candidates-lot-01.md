# Hoja de candidatos Lote 01

## Objetivo del documento

Servir como hoja corta de seleccion para los tres assets mas importantes del primer lote visual:

- `hero-main`
- `services-hero`
- `contact-hero`

La funcion de este documento es comparar candidatos concretos con rapidez, registrar una decision y validar si las tres elecciones funcionan como serie.

## Instrucciones de uso

- completar esta hoja solo con candidatos reales;
- usar una fila por candidato;
- evitar comentarios largos;
- decidir con rapidez si cada opcion pasa a `aprobada`, `descartada` o `revisar`;
- validar la serie completa solo despues de tener al menos una opcion fuerte por asset.
