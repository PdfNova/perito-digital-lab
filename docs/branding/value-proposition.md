# Value Proposition

## Propuesta principal
Investigación digital, análisis forense y trazabilidad tecnológica con enfoque técnico y documental.

## Qué ofrece el proyecto
- análisis de evidencia digital
- reconstrucción y correlación de hechos
- OSINT especializado
- análisis de metadatos y comunicaciones
- informes técnicos y apoyo documental

## Qué no es
- una web de hacking
- un servicio agresivo de “recuperación” o “acceso”
- una marca personal expuesta
- una web centrada en herramientas concretas

## Tono de comunicación
- sobrio
- preciso
- profesional
- claro
- sin exageraciones

## Mensajes clave
- de la evidencia al informe
- análisis, correlación y documentación
- rigor técnico con utilidad práctica
- evidencia digital y trazabilidad para casos complejos
