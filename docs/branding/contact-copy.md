# Copy skeleton - Contacto

## Objetivo de la pagina

Facilitar una primera toma de contacto profesional, filtrar mejor las consultas y dejar claro que tipo de solicitudes encajan en el marco del proyecto.

## SEO provisional

### Title

Contacto para investigacion digital y documentacion tecnica

### Meta description

Pagina de contacto orientada a consultas cualificadas sobre investigacion digital, analisis forense, trazabilidad y documentacion tecnica.

## Hero

### Hero title

Una primera conversacion util empieza con contexto claro

### Hero subtitle

Si el caso requiere orden tecnico, correlacion de indicios o una salida documental mas clara, este espacio sirve para plantear la necesidad de forma estructurada.

### CTA principal

Enviar consulta

### CTA secundario

Ver servicios

### Microcopy del hero

- Consultas mejor orientadas.
- Expectativas claras desde el inicio.

## Estructura de secciones

### 1. Cuando tiene sentido contactar

#### Titulo de bloque

Casos donde una consulta inicial puede aportar valor

#### Texto corto

Tiene sentido contactar cuando existe evidencia digital que necesita contexto, una secuencia de hechos que requiere orden o una necesidad de documentacion tecnica mas clara para terceros.

#### Microcopy

- Evidencia dispersa
- Necesidad de cronologia
- Necesidad de informe claro

### 2. Que conviene preparar

#### Titulo de bloque

Informacion util para una primera consulta

#### Texto corto

No hace falta enviar un relato extenso. Resulta mas util resumir el problema, indicar que materiales existen, describir el momento del caso y explicar que se espera aclarar.

#### Lista sugerida

- breve descripcion del problema;
- tipo de fuentes o materiales disponibles;
- estado actual del caso;
- necesidad principal: analisis, correlacion, trazabilidad o documentacion.

#### Microcopy

- Cuanto mas claro el contexto, mejor la orientacion inicial.

### 3. Solicitudes que no encajan

#### Titulo de bloque

Lo que esta web no ofrece

#### Texto corto

No se atienden solicitudes orientadas a accesos a cuentas, recuperaciones invasivas sobre terceros, identificaciones garantizadas ni peticiones formuladas desde logicas opacas o impropias.

#### Microcopy

- Alcance tecnico y documental.

### 4. Formulario o via de contacto

#### Titulo de bloque

Plantear la consulta

#### Texto corto

El canal de contacto debe invitar a explicar la necesidad con brevedad, claridad y enfoque practico.

#### Campos sugeridos

Campo 1:
Nombre o referencia de contacto

Microcopy:
Puede usarse una referencia profesional o identificador funcional.

Campo 2:
Correo de respuesta

Microcopy:
Se utilizara unicamente para dar continuidad a la consulta.

Campo 3:
Tipo de necesidad

Microcopy:
Seleccionar entre investigacion digital, forense, OSINT, trazabilidad o documentacion.

Campo 4:
Resumen del caso

Microcopy:
Describir el problema en pocas lineas, sin incluir datos innecesarios.

Campo 5:
Materiales disponibles

Microcopy:
Indicar si existen correos, capturas, cronologias, dispositivos, metadatos u otros soportes.

### 5. Nota de prudencia

#### Titulo de bloque

Discrecion y claridad desde el primer contacto

#### Texto corto

La comunicacion inicial debe evitar datos personales innecesarios y centrarse en los elementos tecnicos y documentales relevantes para valorar el encaje de la consulta.

#### Microcopy

- Mejor contexto, menos exposicion.

### 6. Cierre

#### Titulo de bloque

El objetivo del primer contacto no es resolver todo, sino acotar bien el punto de partida

#### Texto corto

Una buena consulta inicial permite valorar alcance, limites y posibles siguientes pasos con mayor precision y menos ruido.

#### CTA principal

Enviar consulta

#### CTA secundario

Ver metodologia

#### Microcopy

- Primer paso: explicar bien el contexto.
