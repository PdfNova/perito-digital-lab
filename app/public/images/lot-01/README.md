# Lote 01

Esta carpeta concentra los tres assets con mayor impacto comercial inmediato:

- `hero-main.svg`
- `services-hero.svg`
- `contact-hero.svg`

## Regla de reemplazo

Cada archivo debe sustituirse manteniendo el mismo nombre y extension para evitar cambios en frontend.

## Criterio minimo

- formato horizontal;
- tono sobrio y premium;
- despacho tecnico realista;
- sin logos ni elementos sensibles;
- coherencia con la serie visual del sitio.
